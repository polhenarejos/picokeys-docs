# Pico FIDO

Pico FIDO is a FIDO2-compatible security device that can be used for passwordless authentication and second-factor authentication.

It supports:

- WebAuthn
- FIDO2 / CTAP
- Modern authentication workflows supported by browsers and operating systems

Pico FIDO is designed to be:

- Simple to use
- Standards-compliant
- Transparent in its security model
