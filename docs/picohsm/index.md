# Pico HSM

Pico HSM is a hardware security module designed to provide strong cryptographic security using a small, auditable hardware platform.

It is designed for:

- Secure key storage
- Cryptographic operations performed inside the device
- Integration with standard interfaces such as PKCS#11

Pico HSM focuses on:

- Minimal attack surface
- Open and inspectable firmware
- Practical security for developers and advanced users
